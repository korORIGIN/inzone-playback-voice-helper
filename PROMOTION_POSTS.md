# Promotion Posts

## 한국어 짧은 소개

INZONE Buds 쓰면서 재생/일시정지/이전/다음 제어가 불편해서 만든 Windows용 무료 도구입니다.  
마이크로 `play`, `pause`, `previous`, `next`만 말하면 현재 재생 중인 미디어를 제어합니다.  
음성 인식은 로컬에서 처리되고, 녹음 파일을 저장하지 않습니다.

다운로드: https://github.com/korORIGIN/inzone-playback-voice-helper/releases

## 한국어 커뮤니티 글

INZONE Buds 사용 중에 손으로 재생 제어하기 애매한 상황이 많아서 작은 Windows 도구를 만들어봤습니다.

기능은 단순합니다.

- `play`: 재생
- `pause`: 일시정지
- `previous`: 이전 트랙
- `next`: 다음 트랙

영어 음성 명령 4개만 인식하고, Vosk 기반으로 PC 안에서 로컬 처리합니다.  
외부 서버로 음성을 보내지 않고 녹음 파일도 저장하지 않습니다.

INZONE Buds 기준으로 만들었지만, Windows 미디어 세션을 쓰는 브라우저/음악 앱이면 다른 마이크에서도 쓸 수 있습니다.

다운로드: https://github.com/korORIGIN/inzone-playback-voice-helper/releases

## English

I made a small free Windows tool for hands-free media control.

It listens for four local voice commands:

- `play`
- `pause`
- `previous`
- `next`

It was built around an INZONE Buds workflow, but it can work with other microphones and Windows media sessions too. Speech recognition runs locally with Vosk. No audio upload, no recording files.

Download: https://github.com/korORIGIN/inzone-playback-voice-helper/releases

## 日本語

INZONE Budsで再生操作が不便な場面向けに、小さなWindows用ツールを作りました。

英語の音声コマンド4つでメディアを操作できます。

- `play`
- `pause`
- `previous`
- `next`

音声認識はVoskでローカル処理され、音声の外部送信や録音ファイルの保存はありません。

ダウンロード: https://github.com/korORIGIN/inzone-playback-voice-helper/releases
