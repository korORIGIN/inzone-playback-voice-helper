# INZONE Playback Voice Helper

비공식 Windows용 로컬 음성 재생 도우미입니다. 영어 음성 명령으로 현재 재생 중인 미디어를 제어합니다.

이 프로그램은 Sony Corporation과 관련이 없는 비공식 개인 프로젝트입니다. INZONE 및 관련 상표는 각 소유자의 자산입니다.

## 명령어

- `play`: 재생
- `pause`: 일시정지
- `previous`: 이전 트랙
- `next`: 다음 트랙

## 사용 방법

1. `INZONE Playback Voice Helper.exe`를 실행합니다.
2. 사용할 마이크를 선택합니다.
3. `듣기 시작`을 누릅니다.
4. `play`, `pause`, `previous`, `next` 중 하나를 말합니다.

## 개인정보

음성 인식은 Vosk 모델을 사용해 PC 안에서 처리됩니다. 이 앱은 음성을 외부 서버로 업로드하지 않고, 녹음 파일을 저장하지 않습니다. 자세한 내용은 `PRIVACY.md`를 확인하세요.
